

## 命令简介

**安装依赖：**

### `yarn install`

## 关于启动

**启动react：**

### `yarn start`

项目链接：[http://localhost:3003](http://localhost:3003)



**启动electron：**

### `npm run electron:start`



**同时启动react+electron**

### `npm run dev`

## 关于打包

**打包react**

### `npm run build`



**打包electron**

### `npm run build-mac`



**同时打包react+electron**

### `npm run pack-mac`
